Theming

Plume is builded with +100 CSS Custom Properties that can be overrided to construct a Custom Theme. You can write your own theme from scratch by overriding classes on your code, or you can use the Theme Editor on the left side of this page.

When using the Theme Editor, you can just tune the inputs to get an imediate visual feedback. Your modifications will be saved on the URL, so you can share your theme, or save its URL as a documentation.

To start experimenting, try to change the App Surface Color to #000000 and the App Font Color to #ffffff and voil√°, you have a dark theme.

You can download your theme by clicking on the "Download Theme" button or by copying its source code on the "Source Code" button. Now you just need to load the theme CSS file on your application

If you prefer to code your theme, or just want to override some custom props, there is a complete reference below. For advanced usage, please read the github repository documentation.

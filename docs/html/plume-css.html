Plume CSS

Welcome to Plume CSS, a lightweight and highly themeable CSS Micro-Framwork. Plume is meant to be the lowest design layer on your application, helping you to grow solid and change easily. Dont wait a huge all-ready-made package, but a complete, friendly, solid and highly customizable design preset to start to code and grow your application with no hassle. Plume was inspired by projects like Skeleton, Milligram, MiniCss and Material, encompassing some practices found in all of them. You can use plume as a lib, a framework or a boilerplate. In this page you will find an interactive theme editor, documentation links and overview, visual reference, and a list of utilities and custom propeties. Custom themes are just a set of CSS Custom properties. You can load them dinamically or scope them to have multiple themes on the same application - even same page.

Typography is based on Material guidelines, and scales with the viewport width. The typography global definitions are pretty easy to tune.
üìê   No grid system
You can easily build any kind of layout with CSS Flexbox and/or Grid. If you still prefer to use a grid system lib, there are a plenty of them avaible out there, there is no need to reinvent the wheel.
üéØ   No bunch of classes
Styles are applyied on pure elements, with just a few utility classes to create variations or helpers. This let you free to create your classes, extend and grow your application as you want.
üí°   Lib, framework or boilerplate
You can use plume as a lib, by adding it on your application as base design. As a framework, by growing from plume, or as a boilerplate, by cloning Plume's source code and starting from there.

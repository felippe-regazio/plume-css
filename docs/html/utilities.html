Utilities

Plume has a large set of utilities that helps you to alter elements styles and behavior. They are pretty useful to avoid code repetition. You can check the complete utilities list on the link below:

Attr Description
.{{prefix}}container Centered padded application container
.{{prefix}}surface Applies the application surface color (--{{prefix}}app-surface-color) as background-color
.{{prefix}}ground Applies the application ground color (--{{prefix}}app-ground-color) as background-color. Useful for divs, card, jumbotrons, etc
.{{prefix}}font-primary Sets font-family to the application primary font (--{{prefix}}font-family-primary)
.{{prefix}}font-secondary Sets font-family to the application secondary font (--{{prefix}}font-family-secondary)
.{{prefix}}padded Applies padding: 16px
.{{prefix}}stacked Applies margin-bottom: 32px
.{{prefix}}disabled Applies disabled style on element. Alias for [disabled], but with no semantic value
.{{prefix}}is-hidden Applies display:none !important style on element. Alias for [hidden], but with no semantic value
.{{prefix}}is-invisible Applies visibility: hidden !important; opacity: 0 !important;
.{{prefix}}no-padding Applies padding: 0px !important
.{{prefix}}no-margin Applies margin: 0px !important
.{{prefix}}no-border Applies border: none !important
.{{prefix}}hgroup Must be applied on a div which holder 2+ Headings. Almost like the deprecated hgroup element
.{{prefix}}dont-print Will hide the element from @print layout

Text

.{{prefix}}h* Applies the heading text style to the given element. For example: .{{prefix}}h1 will apply H1 text style to the given element. You can replace the "h*" for any value on headings range: h1, h2, h3, h4, h5, h6
.{{prefix}}p Applies paragraph text style to the given element
.{{prefix}}mark Applies mark text style to the given element
.{{prefix}}nowrap Forces white-space nowrap on text content
.{{prefix}}text-black Sets color to #000000 by default
.{{prefix}}text-app Sets color to the application font color (--{{prefix}}app-text-color)
.{{prefix}}text-white Sets color to #ffffff by default
.{{prefix}}text-light Sets color to #cccccc by default
.{{prefix}}text-dark Sets color to #444444 by default
.{{prefix}}text-upper Sets text transform to uppercase
.{{prefix}}text-lower Sets text transform to lowercase
.{{prefix}}text-capitalize Sets text transform to capitalize
.{{prefix}}text-center Sets text align to center
.{{prefix}}text-left Sets text align to left
.{{prefix}}text-right Sets text align to right
.{{prefix}}text-primary Sets color to the application primary-color (--{{prefix}}primary-color)
.{{prefix}}text-secondary Sets color to the application secondary-color (--{{prefix}}secondary-color)
.{{prefix}}text-success Sets text-color to the application success-color (--{{prefix}}color-success)
.{{prefix}}text-error Sets text-color to the application error-color (--{{prefix}}color-error)
.{{prefix}}text-warning Sets text-color to the application warning-color (--{{prefix}}color-warning)
.{{prefix}}text-info Sets text-color to the application info-color (--{{prefix}}color-info)
.{{prefix}}link Applies the link style to the text

Buttons

.{{prefix}}button Applies the button style to the given element
.{{prefix}}btn-block Sets display block + width 100% on a button
.{{prefix}}btn-primary Applies the primary button style
.{{prefix}}btn-secondary Applies the secondary button style
.{{prefix}}btn-gradient Applies the gradient button style
.{{prefix}}btn-gradient Applies the gradient button style
.{{prefix}}btn-black Applies the black button style
.{{prefix}}btn-white Applies the white button style
.{{prefix}}btn-light Applies the light button style
.{{prefix}}btn-success Applies the success button style (--{{prefix}}color-success)
.{{prefix}}btn-error Applies the error button style (--{{prefix}}color-error)
.{{prefix}}btn-danger Alias for btn-error
.{{prefix}}btn-warning Applies the warning button style (--{{prefix}}color-warning)
.{{prefix}}btn-round Applies the round button style (--{{prefix}}color-round)
.{{prefix}}btn-text Applies the text button style

Form Controls

.{{prefix}}field Blocked level wrapper for form controls (fieldset inherits this style)
.{{prefix}}label Applies label styles to the given element
.{{prefix}}required Must be applied on labels, .field or fieldsets to add a red asterisk (*) on text
.{{prefix}}input-file Use this class in a span element with an input type file inside for prettify.
[data-{{prefix}}holder] Must be applied on labels when wrapping elements
[data-{{prefix}}subs] Must be applied on range elements to show range max/min as legend
[data-{{prefix}}success] Show a success message on the given element bottom. Use it like data-{{prefix}}success="message"
[data-{{prefix}}error] Show a error message on the given element bottom. Use it like data-{{prefix}}error="message"
[data-{{prefix}}warning] Show a warning message on the given element bottom. Use it like data-{{prefix}}warning="message"
[data-{{prefix}}info] Show a info message on the given element bottom. Use it like data-{{prefix}}info="message"
